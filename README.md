# The Joy of Computing in Python (NPTEL)

This repository contains my solutions and coursework for the "The Joy of Computing in Python" course offered by NPTEL (National Programme on Technology Enhanced Learning).

The course covers the fundamentals of Python programming through a hands-on, problem-solving approach, moving from basic syntax to complex logic and real-world applications.
📌 About the Course

"The Joy of Computing in Python" is an introductory course that focuses on:

    Basic programming constructs (Loops, Conditionals, Functions).

    Data structures (Lists, Tuples, Dictionaries, Sets).

    Application-oriented programming (Cryptography, Image Processing, Game development basics).

    Libraries like random, matplotlib, networkx, and more.

📂 Repository Structure

The repository is organized by weeks to track progress through the course duration:

    Week 01 - Week 12: Each folder contains the programming assignments, weekly practice problems, and scripts developed during the lecture series.

    Assignments: Specific solutions to the graded NPTEL programming assignments.

🛠️ Requirements

To run these scripts locally, you will need:

    Python 3.x

    Recommended libraries:
    Bash

    pip install matplotlib networkx numpy pandas

🚀 Getting Started

    Clone the repository:
    Bash

    git clone https://github.com/sreegiri-x/THE-JOY-OF-COMPUTING-IN-PYTHON--NPTEL-.git

    Navigate to the desired week:
    Bash

    cd "Week 01"

    Run any script:
    Bash

    python script_name.py
